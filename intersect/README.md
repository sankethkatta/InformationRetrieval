# Intersect

A simple Intersection Algoirithm for 2 Lists. The two lists must be in sorted order.
The examples are implemented in:

- [Python](intersect.py)
- [Ruby](intersect.rb)
- [CoffeeScript](intersect.coffee) / [JavaScript](intersect.js)
- [R](intersect.r)
- [Java](intersect.java)
- [Scala](intersect.scala)
